\section{Klasse \code{Extraction}}
Eine Instanz der Klasse \code{Extraction} kapselt alle Regionen~(Methode \code{getRegions}) und Barcodes~(Methode \code{getBarcodes}) die aus genau einem Bild extrahiert wurden.
Jede Instanz tr√§gt, beginnend bei 0, die Nummer des entsprechenden Frames des Quellmaterials~(Methode \code{getFrameNumber}).
Die Barcodes einer Extraction sind in zwei Kategorien unterteilt.
\begin{itemize}
  \item gebundene Barcodes~(Methode \code{getRegionBarcodes}), die von einer Region erfasst sind
  \item ungebundene Barcodes~(Methode \code{getRegionlessBarcodes}), die von keiner Region erfasst sind
\end{itemize}
